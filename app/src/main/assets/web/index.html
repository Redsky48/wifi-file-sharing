    How it works (read once)
    1. On your Windows PC (same WiFi as your phone) run WiFiShareTray.exe. A small circle icon appears in the system tray.
    2. The app finds your phone automatically over mDNS and mounts the shared folder as a network drive (usually Z:\). Open File Explorer — your phone's files are there.
    3. Copy / drag / open / delete files like a normal USB stick. Both directions.
    4. When the phone disconnects (turns off, leaves WiFi, server stopped), the drive auto-unmounts and the tray icon goes gray. Reconnect and it auto-mounts again.

    Requirements

    • Windows 10 / 11
    • .NET 8 Desktop Runtime. If the .exe says it's missing, install it once from microsoft.com/download/dotnet/8.0 (pick "Desktop Runtime", x64).
    • Same WiFi network as your phone (no firewall blocking UDP 5353 for mDNS). Click Allow if Windows Defender prompts.

    If the drive doesn't appear

    • The tray icon's right-click menu lets you pick the device manually and re-trigger mount.
    • Open cmd and run net start WebClient — Windows needs the WebClient service running for WebDAV mounts.
    • For files larger than 50 MB, raise the registry limit: HKLM\SYSTEM\CurrentControlSet\Services\WebClient\Parameters\FileSizeLimitInBytes → set to 0xFFFFFFFF and reboot.

    Under the hood: the phone speaks WebDAV at http://<phone>/dav and Windows mounts it via net use. The tray app only adds discovery, the watchdog, and a clean unmount.
